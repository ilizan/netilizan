node安装

2018-07-18

安装

官网下载node.js，地址。

命令

安装成功后查看node版本

node -v

查看npm版本

npm -v

查看npm安装模块目录

npm config ls

因国内npm较慢，可安装 淘宝镜像cnpm：

npm install -g cnpm --registry=https://registry.npm.taobao.org

查看cnpm版本

cnpm -v

node升级

本地运行 
node -v
npm -v
查看本地node，npm版本。

到node官网下载LTS版，覆盖原安装。

再运行 
node -v
npm -v
查看本地node，npm版本，版本号为下载的node版本号则升级成功。
