vue

全局安装vue 环境 -g：全局 cnpm install -g vue-cli 尝试vue命令： vue 或 vue -V
全局安装：npm install vue-cli -g; 全局卸载：npm uninstall vue-cli -g;
到项目目录下:
# 创建一个基于 webpack 模板的新项目 vue init webpack my-project
提示：项目名，项目描述，作者，是否使用eslint（语法检查工具），不使用。单元测试，不使用，最后一个测试，也不使用。
cd 进入项目文件夹
不能直接运行，需要下载依赖。 dependencies 项目依赖； devdependencies 开发环境依赖；
下载依赖命令：cnpm install
完成后：npm run dev 启动项目

基础

数据渲染：v-test，v-html，{{}}
控制模块隐藏：v-if，v-show
循环渲染列表：v-for
事件绑定：v-on
v-on:click="doFn" 或者 @click doFn 需要再methods下的方法
属性绑定:v-bind 例：class操作
{}对象
[]数组 字符串 直接展示class
总结：
new Vue对象最重要的三个：
data:主要页面展示的数据 methods:方法 watch属性改变后 监听方法
重要指令包括
v-text:渲染数据 v-if:控制显示 v-on:绑定事件 v-for:循环渲染
