vue cli[2.x](建议转3.x版本)

2018-08-21

vue: https://cn.vuejs.org/

vue-cli: https://github.com/vuejs/vue-cli/tree/v2#vue-cli--

安装

首先安装node，安装方法跳转。

常用命令

安装&卸载vue-cli

cnpm install vue-cli -g
npm uninstall vue-cli -g

更新vue版本请卸载后到C:\Users\xxxxx\AppData\Roaming\npm目录下删除缓存文件。

安装后查看（注意大写V）

vue -V

新建vue项目

vue init webpack projectName

安装依赖

cnpm i

启动项目

npm run dev

打包

npm run build



集成bootstrap

一、index.html直接引用相应的css、js

二、集成bootstrap-vuehttps://bootstrap-vue.js.org/

cnpm i bootstrap-vue

main.js里加入

import BootstrapVue from 'bootstrap-vue'
import 'bootstrap/dist/css/bootstrap.css'
import 'bootstrap-vue/dist/bootstrap-vue.css'
Vue.use(BootstrapVue);

项目启动后加入按钮代码测试

<b-button>按钮</b-button>

package.json的dependencies中添加以下依赖项，重新安装依赖时不用单独安装bootstarp-vue依赖。

"bootstrap": "^4.0.0-beta.2",
"bootstrap-vue": "^1.2.0",
"popper.js": "^1.12.9"

具体版本号目前只能用以下命令建本身自带bootstrapVue项目后，进package.json查看。

vue init bootstrap-vue/webpack-simple my-project
