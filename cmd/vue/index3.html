vue cli[3.x]

2018-08-23

vue: https://cn.vuejs.org/

vue-cli: https://cli.vuejs.org/

安装

首先安装node，安装方法跳转。

常用命令

安装&卸载@vue/cli

npm install -g @vue/cli
npm uninstall @vue/cli -g

更新vue版本请卸载后到C:\Users\xxxxx\AppData\Roaming\npm目录下删除缓存文件。

安装后查看（注意大写V）

vue -V

新建项目

方法一

运行图形化界面手动建项目

vue ui

方法二

新建vue项目

vue create hello-world

系统将提示您选择预设。您可以选择基本Babel + ESLint设置附带的默认预设，也可以选择“手动选择功能”以选择所需的功能。

启动

安装依赖

cnpm i

启动项目

npm run serve

打包

npm run build

vue.config.js

因vuecli3精简了结构，没有了复杂的配置文件，所以建议手动新建vue.config.js文件，可配置更多选项。

项目根目录下新建vue.config.js文件

module.exports = {
    // 基本路径
    baseUrl: './',//打包生成index.html里引入cssjs文件相对路径
    devServer: {
        // host: 'localhost', // can be overwritten by process.env.HOST
        port: 8081,//本地开发启动端口
        https: true,//是否开启https
        open: true, //配置自动启动浏览器
    }
}
